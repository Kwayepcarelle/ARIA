# 🎓 ARIA — African Resilience & Intelligence Analytics

> *"We measure what others ignore"*

## 🌍 Description

ARIA est la première plateforme au monde dédiée à l'analyse
de la **charge invisible** que portent les étudiants africains
et son impact sur leur réussite académique.

Là où les autres plateformes mesurent les heures d'étude,
ARIA mesure ce que personne ne mesure :
- 💸 La pression financière familiale
- 🏠 Les responsabilités domestiques
- 🗣️ La barrière linguistique d'enseignement
- 🔦 Les coupures d'électricité pendant les révisions
- 🚌 Le temps de transport quotidien
- 😰 La charge psychologique invisible

## 🧠 Fonctionnalités

| Module | Description |
|--------|-------------|
| 📋 Collecte | Formulaire de collecte de données en ligne |
| 📊 Analyse Descriptive | Statistiques, graphiques, corrélations |
| 📈 Régression Linéaire | Simple et multiple |
| 🔬 Réduction ACP | Analyse en Composantes Principales |
| 🤖 Clustering K-Means | Classification non-supervisée |
| 🌳 Random Forest | Classification supervisée |

## 🛠️ Technologies

- **Python 3.8+**
- **Streamlit** — Interface web
- **SQLite** — Base de données
- **Pandas / NumPy** — Traitement des données
- **Scikit-learn** — Machine Learning
- **Plotly** — Visualisations interactives

## 🚀 Installation
